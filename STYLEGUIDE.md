# The Heat Sheet -- Style Guide

**This file is not public. It is a reference for contributors and AI assistants working on the site.**

## Punctuation

- **No em dashes.** Use `--` (double hyphen) instead of `—` or `&mdash;` everywhere: markdown content, JSX strings, meta descriptions, page titles.
  - Correct: `the essential reference -- a compact, data-rich summary`
  - Wrong: `the essential reference — a compact, data-rich summary`
